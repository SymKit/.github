## 🌟 About SymKit

**SymKit** aims to simplify and accelerate your PHP / Symfony application development. We provide a comprehensive ecosystem of high-quality open-source bundles, designed with premium interfaces and strictly adhering to the best development practices.

Our tools are built for modern developers who want to combine backend robustness with frontend agility.

## 🛠️ Our Principles and Standards

At SymKit, quality is not an option. All our bundles share a strict common foundation:
- **Technical Excellence:** Rigorous code validated by cutting-edge tools (`PHPStan 9`, `GrumPHP`, `Infection`, `Deptrac`, Continuous Integration).
- **Modern Architecture:** Adherence to SOLID principles, extensive use of the `AbstractBundle` and the `Contract pattern`.
- **Developer and User Experience (UI/UX):** Native integration of modern front-end technologies from the Symfony ecosystem: **Tailwind CSS**, **Twig Live Components**, and **Stimulus**.
- **Modularity:** Independent bundles that integrate seamlessly with one another.

👉 *Check out our **[bundles-rules](https://github.com/SymKit/bundles-rules)** repository to discover our AI architecture rules and quality standards.*

## 📦 Our Bundle Ecosystem

Here are the main tools we offer, categorized by use case:

### 🧱 Content & Media
* **[Builder Bundle](https://github.com/SymKit/builder-bundle):** A powerful, strategy-based page-builder system to construct, manage, and render dynamic content blocks.
* **[Media Bundle](https://github.com/SymKit/media-bundle):** Modern media management, performance-oriented with a premium UI/UX.
* **[FAQ Bundle](https://github.com/SymKit/faq-bundle):** Modular FAQ management featuring a CRUD administration interface, drag-and-drop ordering, and Stimulus accordions for public rendering.

### ⚙️ Administration & Operations
* **[CRUD Bundle](https://github.com/SymKit/crud-bundle):** Modern back-office generator with Live lists, filters, sorting, pagination, and Twig components (BackLink, DeleteForm, etc.).
* **[Form Bundle](https://github.com/SymKit/form-bundle):** Premium form types and Twig Live components (RichSelect, Password, Translatable, Slug), all natively styled with Tailwind CSS and Stimulus.
* **[Settings Bundle](https://github.com/SymKit/settings-bundle):** Application-wide settings management via a sectioned UI, automatic cache invalidation, and WebManifest generation.

### 🌐 Routing & Communication
* **[Mailer Bundle](https://github.com/SymKit/mailer-bundle):** Professional email management and sending based on **MJML** templates, featuring event-driven logging and asynchronous sending via Symfony Messenger.
* **[Routing Bundle](https://github.com/SymKit/routing-bundle):** Dynamic database-driven routing offering native performance through seamless integration with Symfony's routing cache.
* **[Redirect Bundle](https://github.com/SymKit/redirect-bundle):** Database-driven URL redirect management (internal and external) with strict validation and global search.
